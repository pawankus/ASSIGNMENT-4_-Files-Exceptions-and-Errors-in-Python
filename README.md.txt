##  Task 1: Read a File and Handle Errors

### Description
This program demonstrates how to:
- Open and read a text file named `sample.txt`
- Print the file content line by line with line numbers
- Handle errors gracefully if the file does not exist

### Features
- Automatically creates `sample.txt` if it does not exist
- Uses `try` and `except` blocks for error handling
- Displays meaningful error messages
